# Sprint 01: The Skeleton Hive

**Goal:** Establish the LangGraph loop and basic "Sovereign Skills" to prevent OOM (Out of Memory) crashes on 1GB RAM.

**Deliverables:**
1. **The Planner:** LLM node that selects from the Skill Registry.
2. **The Skills:** Profiler (low-memory file peeking) and Basic Plotter.
3. **The Monologue:** Chainlit UI showing agent thoughts.
4. **The Sandbox:** Isolated execution of custom Python code.
